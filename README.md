# DA MOTORS Web Platform

A professional, responsive website for **DA MOTORS**, a premium automotive dealership established in 2010. This platform is designed to showcase high-end inventory and provide a seamless contact experience for clients.

## 🚀 Features

* **Responsive Navigation:** Fully adaptive header with mobile menu support and a live wishlist counter.
* **Company Heritage:** Detailed "About Us" section highlighting our mission and decade-long history in automotive excellence.
* **Interactive Contact Hub:** A structured contact form with validation and clear business hour displays.
* **Value Propositions:** A grid-based layout showcasing core services: Premium Selection, Expert Staff, Competitive Pricing, and Quality Assurance.

## 🛠️ Technology Stack

* **HTML5:** Semantic structure for SEO and accessibility.
* **CSS3:** Custom styling (linked via `styles.css`) utilizing a container-based grid system.
* **JavaScript:** Client-side logic (linked via `script.js`) for the mobile menu, wishlist updates, and form handling.

## 📂 Directory Structure

* `index.html` - Landing page (Home)
* `about.html` - Company history and mission
* `motors.html` - Vehicle inventory/catalog
* `contact.html` - Lead generation and location info
* `wishlist.html` - User-saved vehicles
* `styles.css` - Global styles and components
* `script.js` - Interactive functionality
